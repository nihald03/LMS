✅ Auto-deploy on push  
✅ Preview deployments

### Backend Hosting (Render)
✅ Free tier web service  
✅ Auto-scaling  
✅ Automatic redeploy on push  
✅ Environment variables  
⚠️ Cold starts after 15 min inactivity  
⚠️ Limited compute resources  

### Database (MongoDB Atlas)
✅ 512MB storage  
✅ Automatic backups  
✅ Replica set (HA)  
✅ Monitoring tools  
✅ Free forever (no expiration)  
⚠️ Shared cluster (slower)  

### Domain (Optional - Freenom)
✅ Free domain (.tk, .ml, .ga, .cf)  
✅ 12-month renewal  
⚠️ Must renew after 12 months

## ⚠️ Free Tier Trade-Offs

### What You Get (Free)
✅ Production-ready deployment  
✅ Global CDN for frontend  
✅ Cloud database  
✅ Auto-scaling  
✅ SSL certificates  
✅ Monitoring tools  

### What You Don't Get (Free)
❌ No email sending (add SendGrid free tier)  
❌ Backend "cold starts" (first request ~10s after 15 min sleep)  
❌ Shared database cluster (slower than dedicated)  
❌ Limited storage (512MB) - upgradeable  
❌ Limited backend resources  

### When to Upgrade
Upgrade when:
- Your database exceeds 512MB → MongoDB M2 ($57/month)
- Your backend needs more power → Render paid tier ($7+/month)
- You need 99.99% SLA → Enterprise plans
- Your app goes viral! → Scale as needed
